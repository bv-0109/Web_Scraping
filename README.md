# Product Web Scraper

## Objective
This project extracts product details from an e-commerce website using web scraping.

## Features
- Fetch webpage data
- Extract product title
- Extract product price
- Extract product image URL
- Download product image
- Compare product price with target price
- Handle multiple product URLs

## Technologies Used
- Python
- Requests
- BeautifulSoup

## Installation

pip install -r requirements.txt

## Run

python main.py
